## Button Attribute

| 参数     | 说明     | 类型    | 可选值  | 默认值  |
| -------- | -------- | ------- | ------- | ------- |
| label     | 按钮文本 | string  |  -  |  -   |
| primary | 基础样式 | boolean |  -  |  false  |
| size    | 大小 | String | small/medium/large | false   |
| backgroundColor    | 按钮背景色 | String | - | - |
