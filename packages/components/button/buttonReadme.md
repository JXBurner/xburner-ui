## Button Attribute

| 参数     | 说明     | 类型    | 可选值  | 默认值  |
| -------- | -------- | ------- | ------- | ------- |
| type     | 按钮类型 | string  | primary | primary |
| disabled | 是否禁用 | boolean | -       | false   |
| round    | 是否圆角 | boolean | -       | false   |
